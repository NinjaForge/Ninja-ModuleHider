Basic Setup

  • Install the module
  • Disable mootools loading if already loaded by your template or other extensions
  • Enter the module IDs for the modules you wish to display
  • Change the opening and closing messages if desired
  • Decide to place the open/close handle above or below the panel
  • Publish the extension

Tips and Extended Use FAQ

Can I combine multiple modules in one pane?
Yes, simply enter more than one module ID, seperated by commas
Can I have multiple modulehiders on 1 page?
yes, simply copy the module like you would any other joomla module.
How do I change the order the modules appear in?
Simply change the order that they are entered into the Module IDs field.
How do I restrict access to contained modules?
To take advantage of the access level controls, just set the access level for the displayed modules in the module manager screen as you normally would.
Can I use a different Module Suffix?
You can change the module class suffix if you wish to use your template's module styling. We reccomend -flush however, as in 99% of cases it will give the best appearance
How can I style the handle to be more attractive?
Styling the handle can be done in two ways. Firstly by entering HTML directly into the handle text field (such as an img tag), and secondly by using your site's css to style them.

The Ninja Modulehider handle has the following structure to help you lay them out and affect the design: 

<div class="mdlhdr_cont" id="mdlhdr_[this module id]">
  <div class="mdlhdr_hdl" id="mdlhdr_hdl_[this module id]">
  	[Handle message]
  </div>
  <div class="mdlhdr_innr initial_hide" id="mdlhdr_innr_[this module id]">
    [The modules are displayed here]
  </div>
</div>

Trouble Shooting FAQ

What if my site already has Mootools Installed?
Select 'No' for the 'Include mootools script?' parameter if you have another version of the mootools JavaScript library.
My site already has Mootools Installed, but the Modulehider doesn't work!
In order to conserve space and improve performance, Mootools can be downloaded with only the functions needed by a particular developer. Because Ninja Modulehider needs the "slider" functions, if the mootools script your template/component/etc. is loading doesn't contain these functions, then the module hider won't work. In this case we suggest disabling the other mootools script, and loading the one from modulehider. This is because modulehider includes a complete mootools script with all functions, for the widest compatibility.
What if my site has a single combined CSS file?
'Include CSS file', in the advanced parameters tab can be set to 'No' if you have manually entered the css into your own template CSS files. Putting all your css into one or two files will improve your site's load speed.
Wow! My flash banner and shoutbox are all messed up and don't disappear.
Many complex modules, in particular flash ones, render themselves in odd ways which make it difficult or even impossible to hide them. There is unfortunately nothing (we know of) that we can do about it.
